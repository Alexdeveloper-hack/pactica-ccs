Introducción a CSS

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de documentos HTML. Permite separar el contenido de la estructura, proporcionando estilos como colores, fuentes, y layout.

2. Funcionamiento Básico de CSS

CSS funciona aplicando estilos a los elementos HTML a través de selectores. Los estilos se definen en reglas que indican cómo deben aparecer los elementos en la página. Los estilos se pueden aplicar de manera interna, externa o en línea.

3. Incluir CSS en un archivo HTML

CSS se puede incluir en un archivo HTML de tres maneras:

  • Interno: Dentro de una etiqueta <style> en el <head> del documento.
  • Externo: A través de un archivo CSS separado vinculado con una etiqueta <link>.
  • En línea: Directamente en el elemento HTML usando el atributo style.

4. Definición de una regla CSS

Una regla CSS es una instrucción que define cómo debe estilizarse un elemento HTML. Consiste en un selector y un bloque de declaración.

5. Elementos de una regla CSS

  • Regla: Conjunto completo que define el estilo.
  • Selector: Parte de la regla que indica a qué elementos se aplica el estilo (ej. h1, .clase, #id).
  • Bloque de declaración: Contiene uno o más pares de propiedad y valor, encerrados en llaves { }.
  • Propiedad: Aspecto del elemento que se desea estilizar (ej. color, font-size).
  • Valor: Valor específico que se asigna a la propiedad (ej. red, 16px).

6. Medios CSS

Los medios CSS permiten aplicar estilos específicos en función del tipo de dispositivo o condiciones. Se utilizan en consultas de medios @media para adaptar el diseño a diferentes pantallas.

7. Comentarios CSS

Los comentarios en CSS se utilizan para hacer anotaciones y no se renderizan en el navegador. Se escriben entre /* y */.

8. Sintaxis de la definición de las propiedades CSS

La sintaxis general para definir una propiedad CSS es:

selector {
  propiedad: valor;
}

Por ejemplo:

h1 {
  color: red;
  font-size: 20px;
}
